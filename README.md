# Carob Theme
![colorscheme preview](carob.webp "colorscheme preview")

## Currently Supports
- Base16
- btop
- Discord
- Foot
- GTK2/3/4 (limited DE support -- based on [adw-gtk3](https://github.com/lassekongo83/adw-gtk3))
- GTK icons
- Qt/Kvantum
- Mako
- Rofi
- Yazi
